Investigating value memory using visual stimuli in virtual reality environmentat Sharif Neuroscience Laboratory

We recorded subjects’ EEG signals and their performance while playing in a virtual reality environment (implemented by our team for this purpose), in order to investigate value association in humans. Classifying depression patients and normal subjects using EEG signals at Atieh Neuroscience Clinical Center

Evaluation of auditory attention using EEG signals when performing motor and visual tasks at Sharif University Human & Machine Interfaces Lab
Summer 2019

In order to model the human auditory system and evaluate auditory attention while doing visual or motor tasks(using EEG, eye-tracker, and the glove), we defined different tasks so that we could measure the subject’s attention and the brain’s response to unexpected events and also find brain source localization.
